# PNG/JPG → SVG (vtracer)

Simple client-side app using `vtracer-wasm` to convert PNG/JPG to SVG.

Quick start:

1. Install dependencies:

```bash
npm install
```

2. Run dev server:

```bash
npm run dev
```

3. Build for production and deploy to Vercel (or run preview):

```bash
npm run build
npm run preview
```

On Vercel, create a new project pointing to this repo. Vercel will run `npm run build` and serve the `dist` output.
